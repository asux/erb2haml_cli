erb2haml_cli
============

A simple script to convernt all erb files to haml under directory

## Installation

Add this line to your application's Gemfile:

    gem 'erb2haml_cli'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install erb2haml_cli
